BigDataScript

Download and install.

1. Download & Install

Binary distributions are available for Linux and OS.X.

Download Linux Download OS.X

Install

Uncompress the binary at your HOME directory 

cd 
tar -xvzf path/to/bds_*.tgz

BDS's directoy .bds is 'hidden', since the name starts with a dot.

Add BDS's directory to your PATH, by adding the following line to your .bashrc or .bash_profile 

export PATH=$PATH:$HOME/.bds

Mesos (Optional)

If you intend to use bds with mesos, you must install Mesos' native library in $HOME/.bds/lib. For example on a Mac that would be something like 

cp /usr/local/Cellar/mesos/*/lib/libmesos.dylib $HOME/.bds/lib

Requirements

In order to run BigDataScript, you need Java 1.8.

3. Source code

The source code is available at GitHub, here we show how to compile and install.

The source code is available on GitHub . Download the project as follows (you need to install Git first): 

# Clone repository
$ git clone https://github.com/pcingola/BigDataScript.git

In order to complile BigDataScript, you need
  • Java (at least 1.8)
  • Go compiler (at least 1.0)
  • Ant (at least 1.7)
Once you downloaded the project, you can install: 

# Run 'install script
cd BigDataScript
./scripts/install.sh

Then you should add bds to your PATH. Edit your .bashrc (or .bash_profile) and add the following lines 

# BigDataScript
export PATH=$PATH:$HOME/.bds/

Optional: If you have root access, you can do the following (to get bds available everywhere in your system) 

su -
cd /usr/bin/
ln -s /home/your_user/.bds/bds
